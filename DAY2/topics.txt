features of JavaScript:

1. Variables
What is a variable?
→ A name given to the memory location where a value can be stored.

Use of variables
→ To store and manipulate data in programs.

How to declare variables?
→ Using var, let, and const.

2. let vs var vs const
var: function-scoped, can be redeclared and updated.

let: block-scoped, can be updated but not redeclared in the same scope.

const: block-scoped, cannot be updated or redeclared; used for constants.

3. Data Types
Primitive types: number, string, boolean, null, undefined, bigint, symbol

Reference types: object, array, function

4. Operators  
Arithmetic: +, -, *, /, % ,**

Assignment: =, +=, -=, etc.

Comparison: ==, ===, !=, !==, <, > etc.

Logical: &&, ||, !

Unary: ++, --, typeof, !


Ternary: condition ? expr1 : expr2

Bitwise: &, |, ^, ~, <<, >>

5. Conditionals and Nested Conditions
Using if, else if, else

Nesting conditions inside others for complex logic

6. Loops and Nested Loops
for, while, do...while loops

Nested loops for multidimensional iteration

7. While Loop
Executes as long as condition is true

8. Do While Loop
Executes at least once, then checks condition
